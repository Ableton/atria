auto atria::prelude::comp ( F &&  f) -> F&&

Right-to left function composition.

Returns an object g that composes all the given functions \( f_i \), such that

\[ g(x) = f_1(f_2(...f_n(x))) \]

Functions are invoked via standard INVOKE, allowing to compose function pointers, member functions, etc.
